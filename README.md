# CupkekGames VFX MKToon

MKToon shader-backend bridge for `com.cupkekgames.vfx`. Implements the VFX outline / dissolve / shader-property abstractions over MK.Toon shader properties so VFX core stays renderer-agnostic.

## Dependencies

- `com.cupkekgames.vfx`
- `com.cupkekgames.fadeables`
- MK.Toon shader package (asmdef refs, bring your own copy)
